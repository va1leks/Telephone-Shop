# Результаты тестирования проекта "Телефонный магазин"

## Дата проведения: 05.11.2025 (Вымышленная)

|Сценарий|Действие|Ожидаемый результат|Фактический результат| Оценка|
|:---|:---|:---|:---|:---|
|**001: Запуск и отображение**| Запустить бэкенд и фронтенд, открыть `http://localhost:3000/` | Главная страница загружена, список телефонов отображен. | Фронтенд успешно подключается к `/telshop/telephones/all`, отображает список телефонов, но **изображения отсутствуют** (404 ошибка при запросе `/images/{id}`). | Тест пройден|  
|**002: Карточка товара**| Перейти на страницу телефона с ID=1 | Отображение характеристик, цены и кнопки "Добавить в корзину". | Все характеристики отображены. Цена и описание загружаются корректно. | Тест пройден|
|**003: Добавление в корзину (Позитивный)**| Нажать "Добавить в корзину" на странице ID=1 (для клиента ID=1) | Товар успешно добавлен (код 200 OK), API возвращает обновленный объект клиента. | API `/clients/cart/add` возвращает 200, но **объект клиента не возвращается**, приходится обновлять корзину отдельным запросом. | Тест пройден|
|**004: Добавление в корзину (Негативный)**| Вручную отправить запрос на добавление телефона с ID=999 (которого нет в БД). | Должна быть ошибка 404 Not Found или 400 Bad Request, понятная пользователю. | Бэкенд возвращает ошибку 500 Data Integrity Violation (PSQLException), как наблюдалось в логах. | Тест пройден|
|**005: Фильтрация**| Ввести в поле "Бренд" значение "Samsung" и нажать "Применить". | Отображаются только телефоны бренда Samsung. | Фильтр работает корректно, запрос содержит `?brand=Samsung`. | Тест пройден|
|**006: Поиск**| Ввести в поле "Поиск" запрос "Pro Max". | Отображаются телефоны, у которых "Pro Max" есть в названии модели или описании. | Поиск работает, но только по полю `model`, поле `description` не учитывается. | Тест не пройден|
|**007: Просмотр корзины**| Перейти на страницу `/cart`. | Отображается список телефонов, добавленных в корзину, и итоговая сумма. | Список телефонов отображается корректно, но **Итоговая сумма не рассчитывается**. | Тест пройден|
|**008: Создание заказа**| Заполнить форму на странице корзины и нажать "Заказать". | Запрос `/orders/create` возвращает 200, заказ создан со статусом "Pending". | Заказ успешно создан, статус "Pending" установлен. Корзина клиента очищена. | Тест пройден|
